## Why

Private fields and symbols answer different questions: one enforces encapsulation, the other creates collision-resistant public keys.

## What Teaches

This lesson teaches private fields, symbol-keyed state, encapsulation vs reflection, API surface vs implementation detail, debugging and testing tradeoffs.
