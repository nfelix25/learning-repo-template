## Why

A symbol becomes useful when it is used as a key that ordinary string-based code will not accidentally collide with.

## What Teaches

This lesson teaches computed property names, symbol-keyed object literals, descriptors for symbol properties, public but non-colliding extension points.

## Prereqs

`02-property-keys-and-object-internals`

The test suite in `lessons/04-symbol-keyed-properties/symbols.test.ts` passes.
