## Why

Generics are the foundation of reusable TypeScript. Without them, you choose between losing type information (with `any`) or duplicating code for each type. This lesson establishes the syntax and inference rules you'll use in every subsequent lesson.

## What Teaches

Generic function and class syntax; multiple type parameters; generic interfaces; when TypeScript infers type arguments vs when you must annotate them explicitly.

## Prereqs

None.

## Success criterion

The test suite in `lessons/01-generic-fundamentals/generics.test.ts` passes.
